Restifier

restify~ Restifier

restify.js:138

new Restifier()

Used to "restify" a sequelize model. Adds methods like get() and post() to the model, along with restify(), which returns an object with various properties (most important is .paths, a dictionary of functions keyed by http method (eg. .get()) returning descriptions express can consume for each http method key (eg. "${prefix}/todo/:id").

Instances of the model also get a .restify() method with paths which, for the instance, return the specific path the instance resides at (eg. "${prefix}/todo/1").
